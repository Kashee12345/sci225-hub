Which Lipid Does Each Agent Actually Move?

A lipid panel is four numbers, and no drug moves all four. Each agent reaches into one place — the gut, the liver, the capillary — and what it can grab there decides which row on the panel changes. Tap a drug class to run it on this patient; tap a row of the panel to see who moves that number.

The panel & the pathway

Grey tick = the goal. Dashed tick = where this patient started. Values are a typical single-agent response, not a promise — real percentages vary with dose, drug, and patient.
