A count is binomial only if all four hold: the number of trials n is fixed in advance · each trial is independent · each trial has exactly two outcomes · and p is the same on every trial.

Does it actually happen that way?

Each "experiment" runs 10 independent trials and records how many successes came up. The gold bars are what really happened; the blue bars are what the formula predicted.
