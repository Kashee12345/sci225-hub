Types of Long-Term Memory

Long-term memory is not a single vault. It divides first into explicit memory — the facts and events you can consciously declare — and implicit memory, the skills and reflexes your body performs without words. Explicit splits again into episodic (events you lived) and semantic (facts you know), while procedural memory anchors the implicit side.
