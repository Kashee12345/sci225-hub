Where Did the Toy Go? Object Permanence

To an infant, a hidden object may simply cease to exist. Understanding that things keep existing when you can't see them — object permanence — unfolds gradually across the six substages of Piaget's sensorimotor stage (birth–2 years). Pick a substage, then run the hide-and-seek test and watch how the baby responds.
